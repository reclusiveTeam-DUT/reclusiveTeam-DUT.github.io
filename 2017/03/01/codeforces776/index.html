C.Molly’s Chemicals
Description: 给一个数列a和一个数k，求a的子列中和恰好为k的次方的子列的个数。
Analysis:
cj: 注意数据范围，和最大是1e14，k最大是10。维护前i项的和sum，用cnt[x]记录sum == x的个数。每次答案+=cnt[sum - k ^ j]，直到超出界限。单独处理一下 k == 1 和 k == -1的情况。

D.The Door Problem
Description: 有很多门，给定开始的状态，开或关。有很多开关，每个开关控制着一些门。每一个门都被两个开关控制。问能不能让全部的门都被打开。
Analysis:
cj: 初始状态开着的门只能被操作一次，所以两个开关中只能选一个。初始状态关闭的门能被操作两次或零次，所以两个开关必须一起选择我或者一起不选择。用并查集维护以下就好了，[1, m]表示用这个开关，[m + 1, 2m]表示不用这个开关。
